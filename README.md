# SpringToyProject_1
안녕하세요, JAVA Spring을 이용한 Toy 프로젝트입니다.

### 개발 환경
- Spring Boot
- Spring Data JPA
- Spring Security
- Gradle with Intellji
- Junit5
- Spring Web
- Thymeleaf
- Lombok
- H2 Database / Mysql


### 소개
- API 통신을 중점적으로 설계
- 자바의 순수 HttpSession 을 이용하여 로그인 서비스 구현 (추후 스프링 시큐리티 적용 예정)
- Junit5 테스트 코드를 중요시
- 간결한 코드, 누구나 알아볼 수 있는 코드, 버그 없는 코드를 목표로
- 예외 Exception 처리 깔끔하게
